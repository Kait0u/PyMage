# PyMage
Python application to manipulate images
Credit project for Image Processing Algorithms class 

# Features

## Class 1
- Image read
- Color conversions
- Histogram operations
- Unary and Neighborhood operations

## Class 2
- Image arithmetics
- Convolution using filters

## Class 3
- Morphology Ops
- Image Pyramid
- Profile Line

## Class 4 (WIP)
- Segmentation
  - Thresholding
  - GrabCut
  - Watershed
- Inpainting
- Object Traits
- RLE (WIP)
